# AI 과제 유형별 성능평가 지표

- 모델을 만들면 모델에 대해 해석할 수 있어야 한다. 그래야 어떤 모델이 더 좋은지 판단할 수 있다.
- 결과에 대한 해석을 위해 성능평가 지표에 대해 알아보자.

회귀 모델의 성능평가 지표

- MSE, MAE

분류 모델의 성능평가 지표

- Accuracy
- Recall
- Precision

# 딥러닝 활용 (이미지 딥러닝)

- 텍스트 딥러닝
  - RNN(Recurrent Neural Network) Model
- 이미지 딥러닝
  - CNN(Convolutional Neural Network) Model

- 이미지 딥러닝 처리를 위해 이미지의 구조에 대해 알아보자.
  - RGB 세 개의 채널을 가진 array 형태
  - 각 픽셀은 0~255로 이뤄짐.

FC layer(Fully Connected)

- 완전 연결 계층
- 한 층의 모든 노드(뉴런)이 다음 층의 모든 노드와 연결된 상태
- 이미지에서 FC 레이어를 사용할 경우, 이미지가 조금만 커지더라도 계산해야 할 가중치가 매우 커짐. 연산량 증가.
- 또한, 이미지의 공간적 정보도 잃는다.

Convolution layer

- FC layer의 공간적 정보 상실 문제를 해결하는 방법
- 공간정보를 압축
- 필터 개수를 늘려서 여러 특징을 추출하자. 필터는 Node 역할
- 필터의 개수만큼 특징 추출
- convolution layer(이하, conv)가 한 칸씩 움직이면서 이미지를 훑으면서 Activation Map을 만든다.
- stride 옵션:
  - 기존 한 칸씩에서 n 칸 이동으로 변경 가능. 빠르게 훑기 위한 옵션

Pooling layer

- Max Pooling: 가장 큰 값만 특징으로 추출

## CNN 모델

- conv와 maxpooling을 뱐복하여 이미지를 압축
- 마지막에 flatten(평탄화)으로 이미지를 다시 넓게 펼침
- 이 사이에 표준화, 정규화와 같이 스케일링 과정이 필요할 수도 있음
- 배치 정규화(batch normalization)라는 필터를 통해 표준화를 할 수도 있음.
- dropout 등을 통해 오버피팅을 방지가 필요할 수도 있음.

# 마무리

1. 딥러닝을 활용한 알고리즘에는 RNN, CNN 등이 있다.
2. 이미지는 배열의 형태로 각 픽셀 값은 숫자 0~255로 이뤄져 있다.
3. CNN에서는 Convolution layer filter로 공간정보기반 특징을 추출한다.
4. Convolution layer는 stride 옵션으로 빠르게 훑을 수 있다.
5. Pooling layer로 데이터를 요약한다.
6. 주로 Conv, Maxpooling을 섞어 사용하여 CNN을 구성한다.
