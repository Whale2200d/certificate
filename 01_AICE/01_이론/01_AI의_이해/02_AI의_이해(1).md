# 자동화와 AI의 차이

- 자동화: Process-driven, 규칙 기반 작업, process를 사람이 지정, input과 process를 컴퓨터에게 제공

- AI: Data-driven, 데이터 기반 작업, data 기반 머신이 스스로 process를 찾아가는 과정, input과 output을 컴퓨터에게 제공

# AI를 배워야 하는 사람

- AI 알고리즘 개발자
- 데이터 사이언티스트
- 분야별 전문가

# AI 기본 이론 - 머신러닝

- 머신러닝이 도입되기 전에는 사람이 컴퓨터에게 기준 하나씩 프로그래밍으로 지정함.
- 머신러닝이 도입된 이후에는 사람의 개입 대신 컴퓨터가 스스로 기준을 찾아 나감.
- 대표 예시: 스팸 메일 분류

## 예시

**자동화 예시**

규칙: 실내온도가 25도 이상이면 에어컨 켜기

데이터: 실내 온도 데이터

&rarr; 해답: 에어컨 on/off

**머신러닝 예시**

데이터: 실내 온도, 외부 온도, 습도 데이터

해답: 온도/습도 데이터별 에어컨 on/off 여부 데이터

&rarr; 규칙: 에어컨 on/off 방정식 생성

## 머신러닝 학습 유형

- 머신러닝은 지도학습, 비지도학습, 강화학습로 구분
- 강화학습은 Basic에서 배우지 않음

- 지도학습(Supervised Learning): 정답(label)을 알려주면서 진행하는 학습. 데이터와 정답이 함께 제공됨. (라벨링 작업 선행 필요)
- 비지도학습(Un-Supervised Learning): 정답(label)이 없이 진행되는 학습. 데이터 자체에서 패턴을 찾아내야 할 때 사용함.

## 지도학습 이해하기

실무에서 가장 많이 사용. 비교적 이해하기 쉬운 머신러닝 학습 방법

테이블에서:

- 행(row): 개체(instance), 관측치(observed value), 기록(record), 사례(example), 경우(case)
- 열(column): 특성(feature), 속성(attribute), 독립변수, 필드(field)
- 타겟 변수: 종속 변수, 타겟, 라벨

지도학습에 대한 데이터는 피처(독립 변수)와 타겟(종속 변수)로 구성되어 있어야 한다.

지도학습으로 해결할 수 있는 문제는 분류(Classification), 회귀(Regression)가 있다.

- 분류는 범주 예측. 합격/불합격, A~E 등급 구분 등
- 회귀는 수치 예측. 시험 점수, 주가 등

(⚠️ 주의사항: 카테고리인지, 수치인지를 확인하는 것이 포인트. 숫자면 무조건 회귀라는 판단은 지나침. 예시: 타겟 변수가 1, 0일 경우, 숫자이지만 카테고리를 예측하는 문제라 볼 수 있다.)

분류와 회귀의 구분은 AI 모델의 방향성을 잡는 시작이 된다. 초기에 잘 배워야 한다.

- 분류 예시: 합격 여부(합/불합), 악성 종양 여부(양성/음성), 소고기 등급(1++, 1+, 1, 2, ...), 오지선다(1~5)
- 회귀 예시: 시험 점수(10, 17, 20, ...), 판매량(88, 56, 121, ...), 집값(12억, 21억, 5.5억, ...), 키(178.3, 169.5, 184.7, ...)

(⚠️ 주의사항: 분류, 회귀 헷깔리는 경우가 생기기 마련. 시험 점수의 경우 애매한 사례임. 이 경우, 둘 다 만들고 성능 잘 나오는 것 사용하는 방법이 적합함.)

## 데이터 분할(data split)

전체 데이터는 학습 데이터(70%), ~~검증 데이터(10%)~~, 평가 데이터(20%)로 구분하여 사용한다.

모델 구조 측면에서 데이터 분할 살펴보기

1. 수집된 데이터 (Feature, label)를 학습(70), 평가(20)으로 구분
2. 학습 데이터로 AI 모델 학습
3. AI 모델에 평가 데이터의 feature 인풋하여 예측 결과 확인
4. 예측 결과 vs 실제 결과(평가 데이터의 label) 비교하여 모델 성능 평가

학습 데이터 분할하기

- 배경: 메모리의 한계와 속도 저하를 막기 위함
- 학습 데이터를 batch(베치)라는 작은 단위로 쪼개서 학습한다.
- 1회 학습을 마치면 1 epoch(에포크)가 완료되었다고 한다.
- batch가 1 epoch 종료까지 횟수(개수)를 iteration이라고 한다.
- batch size \* iteration = 1 epoch
- 예시
  - 수집 데이터가 20,000개 라면, 학습 데이터는 14,000개
  - batch size가 128일 때, iteration은 약 110번
  - 이는 110번 학습했다고 볼 수 있다.
